📱 앱 설치하기
📋 1단계: 앱 다운로드
1. 위의 "앱 설치하기" 버튼을 탭하세요
2. "설치" 확인 대화상자가 나타나면 "설치"를 선택하세요
3. 홈 화면에 앱 아이콘이 나타날 때까지 기다리세요
🔐 2단계: 개발자 인증서 신뢰 (필수!)
앱이 다운로드되었지만 실행되지 않는다면:

iPhone/iPad에서:
1. 설정 → 일반 → VPN 및 기기 관리
2. Enterprise 앱 섹션에서 개발자 찾기
3. 개발자 이름을 탭하고 "신뢰" 선택
4. 확인 팝업에서 "신뢰" 다시 선택

⚠️ 이 단계를 완료해야만 앱이 실행됩니다!
📱 3단계: 앱 실행
인증서 신뢰 완료 후 홈 화면의 앱 아이콘을 탭하여 실행하세요
🔧 문제 해결:
• 앱 아이콘이 회색으로 표시: 인증서 신뢰 미완료
• "신뢰할 수 없는 개발자": 위 2단계 다시 확인
• 앱이 삭제됨: 다운로드부터 다시 시작
• 설치 버튼이 동작하지 않음: Safari 브라우저 사용 확인
